下载和教程-linux 使用教程
=====================================================
执行 cd && mkdir clash 在用户目录下创建 clash 文件夹。

下载适合的 Clash 二进制文件并解压重命名为 clash

一般个人的64位电脑下载 clash-linux-amd64.tar.gz 即可。
下载地址：https://github.com/Dreamacro/clash/releases
下载解压后得到文件：clash-linux-amd64   没有后缀。直接运行的。
======================================================
2.
在终端 cd 到 Clash 二进制文件所在的目录，执行 wget -O config.yml https://sub.sdncimcin.xyz/你的订阅地址.info   下载 Clash 配置文件
下载完配置文件记得改名为：config.yaml 与clash-linux-amd64 放到一个文件名，重要。
=========================================================================
执行 ./clash -d . 即可启动 Clash，同时启动 HTTP 代理和 Socks5 代理。

如提示权限不足，请执行 chmod +x clash     
注意后面有一个点的，  ./clash -d . 点。
=====================================================================
http://clash.razord.top/
访问 Clash Dashboard 可以进行切换节点、测延迟等操作。

Host: 127.0.0.1，端口: 9090

打开   http://clash.razord.top/  进入管理界面。
==================================================
5.
以 Ubuntu 19.04 为例，打开系统设置，选择网络，点击网络代理右边的 ⚙ 按钮，选择手动，填写 HTTP 和 HTTPS 代理为 127.0.0.1:7890，填写 Socks 主机为 127.0.0.1:7891，即可启用系统代理。


因我的debian 没有网络设置的界面，用 chrome Proxy SwitchyOmega 解决了网页代理。系统代理暂时没有解决方案。
==============================================================
